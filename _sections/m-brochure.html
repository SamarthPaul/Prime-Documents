PR-MT-003

Brochure

A Marketing Tools framework that captures an entrepreneur's brochure / catalogue content — setup details, the list of products featured in the brochure (with briefs, dimensions, price, MOQ and product pictures) and the draft-by-draft design status — and tracks the Designer's involvement from data collection through entrepreneur approval and printing.
Key behaviour. The Brochure Type chosen at setup is a multi-select; the same set of types is mirrored down into the Product Listings child table so each product can be tagged with the brochure type(s) it should appear in. A product is considered “included” in a brochure only when at least one Brochure Type is selected on its listing row. Design work runs through up to three drafts; the brochure is final only when Approved by Entrepreneur is ticked.

On-Screen UI Structure

A teal title bar reads Brochures Module with crumb PRIME Rural · Marketing Tools. Below it a horizontal tab rail — Summary · Existing · Required · Resources · Task Log — with the active tab highlighted teal.

Summary is read-only: a Brochure-wise Overview table (one row per brochure, with the two right-most columns — Implementation Status and Approved — shaded terracotta/green), four aggregate cards (Existing / Cost / Interventions / Status) and a four-tile Intervention Flags strip.

Existing opens with an Entrepreneur chip (Entrepreneur Name + Enterprise Name, both auto-fetched from profile, shown italic on the cream-deep background), followed by the Existing Brochure child table where each current brochure/catalogue is listed, its file uploaded, and a Need New Brochure? toggle plus justification recorded.

Required is split into two banner-blocks. Brochure Details (teal banner) holds the Brochure Setup form (enterprise, multi-select Brochure Type, Market Scope, tagline, AI-assisted storyline, about, contact, email, address) and a nested Product Listings child table — the brochure's product listing grid carrying per-product brochure type(s), product name, brief, dimensions, price, MOQ and a multi-file product picture upload/view cell. Brochure Status (terracotta banner) holds Designer, Print Format and Page Count fields and the draft-tracking child table (Draft 1/2/3 file uploads, each with a status dropdown, plus the Approved-by-Entrepreneur checkbox).

Resources is a placeholder for brochure templates, design references, print specifications and brand guidelines. Task Log is a numbered activity grid driven by a fixed 12-item brochure task list.

Calculated / read-only fields (Entrepreneur Name, Enterprise Name, every Summary metric) render italic on the cream-deep background and are never editable; each Summary roll-up carries a derivation tooltip (Rule 1).
